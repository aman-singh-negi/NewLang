# AI Datasets

Purpose:
- Store generated datasets for correct/incorrect/optimized programs.
- Provide repeatable data generation for AI training and experiments.

## Generator
- Script: `generate_datasets.py`
- Output files:
  - `correct_programs.json`
  - `incorrect_programs.json`
  - `optimized_pairs.json`

## Usage
- Run from this directory:
  - `python generate_datasets.py`

This generates 100 samples per category by default.
